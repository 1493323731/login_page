# login_page

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
### 项目描述
一个精美的登录页面，拥有登录、注册、找回密码、重置密码等功能  

### 技术栈
vue3,vue route,vue cli,axios,element-plus,typescript
